## 🚀 Projetos em destaque

### [NYC Taxi Snowflake Pipeline](https://github.com/hugo21234/nyc-taxi-snowflake-pipeline)
Pipeline end-to-end sobre o dataset público NYC Yellow Taxi com arquitetura Medallion (Bronze → Silver → Gold). Ingestão automatizada em Python, transformações em SQL e Gold layer pronta para consumo BI.
`Python · Snowflake · SQL · Arquitetura Medallion`

### [Pipeline Olist — Diagnóstico de Saúde de Sellers](https://github.com/hugo21234/Pipeline-de-Dados-Olist-Diagn-stico-de-Sa-de-de-Sellers)
Pipeline analítico sobre o dataset real Olist (e-commerce brasileiro) para diagnóstico operacional de sellers: atraso de entrega, ticket médio e reputação. Modelagem em Python + SQL com foco em insights acionáveis.
`Python · SQL · Pandas · Análise de Dados`

### [Projeto-Pessoal](https://github.com/hugo21234/Projeto-Pessoal)
API REST modular com autenticação JWT, Bcrypt, Express 5 e PostgreSQL via Sequelize. Arquitetura em camadas orientada a produção.
`JavaScript · Express · PostgreSQL · Sequelize · JWT`

### OrganizeMais — Plataforma Financeira via WhatsApp
Agentes de IA generativa que interpretam linguagem natural, categorizam transações e geram relatórios financeiros autônomos via WhatsApp. Backend em Supabase com RLS, Edge Functions e orquestração via n8n.
`Supabase · n8n · OpenAI · Evolution API`

### Ayron — Automação Operacional de Clínica Médica
SDR automatizado completo: qualificação de leads, follow-up por estágio, agendamento e sincronização em tempo real entre todos os setores via n8n.
`n8n · OpenAI · Supabase · ClickUp`
